# Philosophy Foundation

## 这个项目是干嘛的

这是一个用来训练我思考能力的项目，目标是建立一套稳定的价值判断系统，而不是简单记录知识。

## 为什么我要做这个项目

我在做选择时没有稳定的判断标准，只能依赖外界的反馈（比如钱、流量、表面成果）来判断事情是否值得做，这会导致我在看到别人有结果时产生动摇。

我希望通过这个项目，逐步建立一套不依赖外界噪音的判断方式。

## 这个项目不是什么

这个项目不是：

- 情绪发泄口
- 摘抄哲学名词的地方
- 用来让自己“看起来在思考”的工具

所有内容必须围绕真实问题，而不是抽象表达。

## 当前阶段

Day1：识别自己在判断和选择上的不稳定问题

## 学习路线

当前路线：

1. 现实问题清单 +《天道》片段：激发问题意识，但不把电视剧当真理
2. 逻辑学基础：训练判断，防止被话术和概念带跑
3. 斯多葛主义：区分可控与不可控，稳定情绪
4. 实用主义：把想法落实到行动和结果
5. 伦理学基础：防止变得冷漠、功利、傲慢
