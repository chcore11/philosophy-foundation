# 1. 我为什么要学哲学

我发现自己在一些具体情况下会不稳定，比如：

- 做项目的时候看到别人搞ai有所成果而且似乎能赚好多钱，我会怀疑自己方向有没有用，能不能让自己在以后持续复利
- 有时候会拖延或者被短期欲望带走，我希望去思考这一套底层逻辑，以后遇到问题时可以尽快摆脱
- 同时自己容易被身边人影响，看到别人搞自媒体挺有流量，看到别人搞ai+某领域看起来有模有样等等，自己的情绪会发生波动

所以我感觉自己缺一个更稳定的思考方式。

# 2. 我现在缺什么

我目前的问题不是“什么都缺”，而是：

- 在做选择时没有明确标准，即判断一件事情值不值得做
- 容易被情绪或者外界影响，缺乏一个强大的基本盘
- 对长期方向有模糊但不清晰，当前在做嵌入式，眼光不够长远，只看到当下做基于g474re的agv闭环控制

# 3. 我希望哲学帮我解决什么问题

我希望哲学能帮我做到：

- 在做决策时有一套自己的判断标准
- 面对不确定性时不轻易动摇
- 能区分什么是短期欲望，什么是长期有价值的事情

# 4. 下一步行动

这份记录先保留最初的问题，不急着给出完整答案。接下来：

- 选择一个当前项目，写清楚它解决什么问题、训练什么能力、四周后留下什么成果。
- 下次因为别人的成果而动摇时，记录我具体羡慕什么，而不是只写“方向不稳定”。
- 每周做一次复盘，检查本周的思考是否改变了实际安排。

底线：不能用“我在学习哲学”替代项目推进和现实选择。
